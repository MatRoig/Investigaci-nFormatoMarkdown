# InvestigaciónFormatoMarkdown

Conceptos básicos para investigar

Qué significa la extensión de un archivo

La extensión de un archivo es un indicador que acompaña al nombre del archivo y que indica a la computadora el tipo de información que contiene este, para que así esta pueda leerlo con el programa adecuado. Nos ayuda a identificar y organizar nuestros archivos como usuarios. Algunos ejemplos serían .txt .exe .docx

Qué es Markdown

Es un lenguaje para dar formato a texto a través de un código de caracteres. Este fue diseñado para poder crear textos para subir a la web de forma más eficiente que usando HTML. A su vez también es una herramienta de software, ya que podemos traducir este lenguaje más sencillo a HTML usando programas dedicados a esto o en entornos que soporten Markdown.

Para qué se usan archivos .md

Es la extensión de un archivo markdown. Solo contiene texto y se puede abrir con cualquier editor.
